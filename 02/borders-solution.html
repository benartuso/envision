Important note on these solutions:

As I'm sure you'll notice, we're putting our CSS in the "style" tag in the header, rather than in an external CSS file for these solutions. This is for your convenience, to keep all of the code for this relatively short example in one place.

If you wanted to link to an external stylesheet, you'd do it with the following tag in the header:

<link rel = "stylesheet" href = "path-to-your-stylesheet">.
