Style, three ways!

Method 1: Inline style

Not a best practice! A worst practice, tbh.

(Specifically, the reason this is such a bummer is because it causes us to write a ton of redundant code that isn't easily changed. To change the styling of the paragraphs below, we'd have to write a billion individual lines of CSS, rather than using the power of selectors to change only one.)

This folder contains three copies of approximately the same HTML document.

These documents will contain the same basic information, and will even be styled the same way with CSS, but will have those styles applied in three different ways to illustrate the different ways of applying CSS.

So, let's get to it! The basic style rules to test out for this document:
  • The font of the entire body will be Helvetica, instead of the default Times New Roman.
  • Specific text elements can be decorated with colors, such as purple, blue, and coral. These elements will also be made slightly larger than the rest of the document and occupy a heavier text weight.

All paragraphs will be wrapped with a 2px solid blue border, with 5px padding on all sides.

Some paragraphs, though, such as the next one, will instead have a 5px dashed green border, with 10px padding on all sides.

This is the paragraph that's unique and funky.
